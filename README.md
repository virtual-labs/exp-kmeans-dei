### About the Experiment 

This experiment focuses on performing clustering using the K-Means algorithm and determining the optimal number of clusters using the Elbow method. It covers customer segmentation based on purchasing behavior using Annual Income and Spending Score features, along with model evaluation using Silhouette Score, Davies-Bouldin Score, and Calinski-Harabasz Score.
